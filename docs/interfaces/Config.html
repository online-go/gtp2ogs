Optional ending_bot

ending_bot?: EndingBotConfig

Secondary bot to use for ensuring your real bot passes or resigns appropriately. This bot will be consulted every move after . If the move it returns is a pass it will override the move your bot has made (unless your bot is resigning). If the move it returns is a resign, it will count the number of successive resigns and if it is more than the number of allowed resigns you've set in this config (default of 3), it will override your bot's move with a resign.
